# vl-checkbox
De checkbox laat de gebruiker toe om een of meerdere opties te selecteren uit een lijst. Gebruik de checkbox in formulieren.

## Installation
```
npm install --save vl-ui-checkbox
```

## Demo
```
npm run demo
```
